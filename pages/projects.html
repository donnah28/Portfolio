Simon Game

❮ ❯

Bootcamp project; the simon game comes from the basis of simon says. When the web page loads up it will ask you to press any key; when pressed the game will begin. I used Javascript to get a randomised sequence of the buttons lighting up. The user will then have to remember the sequence and press the same buttons in the same order.

The levels will then move up adding another button to the sequence each time. If the user presses the wrong button at the wrong time, a game over screen will show (second image in slideshow) and the game will restart

Skills used: HTML, CSS, Javascript

Drum Kit

Drumkit was a bootcamp project, where I used html, css and javascript to create a drumkit that can create sounds either from using the mouse to press each drum or using the keys on the keyboard.
